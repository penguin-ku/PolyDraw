Description

Polygonal modeling is an approach for modeling objects by representing or approximating their surfaces using polygons. Polygonal modeling is well suited to scanline rendering and is therefore the method of choice for real-time computer graphics. Alternate methods of representing 3D objects include NURBS surfaces, subdivision surfaces, and equation-based representations used in ray tracers.

PolyDraw is a free 3D polygonal modeller, for creating or modifying 3D objects using a mesh of 3D points and parametric NURBS Curves .Exports and imports to over 40 formats including WebVR and 3D Printing.

Features

WebVR A-Frame

A-Frame is an open-source web framework for building virtual reality (VR) experiences. It is primarily maintained by Mozilla and the WebVR community. It is an entity component system framework for Three.js where developers can create 3D and WebVR scenes using HTML.

STL for 3D printing

Stereolithography (STL) is one of the most common file types that is used for 3D printing. Unlike material removed from a stock in the conventional machining process, 3D printing or AM builds a three-dimensional object from computer-aided design.

JSON for ThreeJS

Three.js allows the creation of GPU-accelerated 3D animations using the JavaScript language as part of a website without relying on browser plugins. High-level libraries make it possible to author complex 3D computer animations that display in the browser.

RGB Colour Picker

The RGB color model is an additive color model in which red, green and blue light are added together in various ways to reproduce a broad array of colors. The name of the model comes from the initials of the three additive primary colors, red, green, and blue.

Geometric primitives

Constructive primitives are simple geometric shapes such as a cube, cylinder, sphere, cone, pyramid, torus. Modern systems may operate with primitives which are lines (segments of straight lines, circles and complicated curves), and shapes (boxes, arbitrary polygons, circles).

Over 40 formats

PolyDraw supports Import and Export between more than 40 formats and also can be used as a standalone 3D converter besides beeing an editor. All the most common 3D formats are supported and can be edited as the user wishes before export as desired.
